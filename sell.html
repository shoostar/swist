Instead of precompiling math for our grids we can do it live in the browser using CSS variables, thus saving us a LOT of lines in CSS (...in the future. I'm looking at you Explorer). This is achieved using nothing but vanilla CSS.

All of the text you see here is also responsive, based on viewport units. It moderately scales with the browser window, giving a proportionately smaller font-size/leading on small screens, larger on big screens.
